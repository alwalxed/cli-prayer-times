# CLI Prayer Times

A command-line tool for displaying prayer times.

## Installation (NPM)

```bash
npm i -g cli-prayer-times
```

## Usage

#### Basic Command

```
prayers
```

#### Change City

```bash
prayers -c

## you will be prompted to enter a city name,
## and the prayer times for that city will be displaye
```
